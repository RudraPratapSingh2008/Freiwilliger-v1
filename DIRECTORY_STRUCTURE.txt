KEY CONFIGURATIONS:

CLIENT ROUTING:
  / → HomePage
  /login → LoginPage
  /register → RegisterPage
  /dashboard → DashboardPage
  /settings → SettingsPage

API BASE URL:
  Client: http://localhost:5000/api/v1
  Proxy configured in vite.config.js

SOCKET.IO:
  Server: http://localhost:5000
  Client connection on auth with JWT token
  Personal room: user:{userId}

REDUX STATE:
  auth.user → Current logged-in user
  auth.accessToken → JWT access token
  auth.isAuthenticated → Login status
  auth.loading → API loading state
  auth.error → Error message

AXIOS INTERCEPTORS:
  Request: Adds Authorization: Bearer {token}
  Response: Auto-refresh on 401 status
